Usage:
  fhpack {-c|-d} [-h] [-1|-9] infile outfile

  fhpack {-t} [-h] [-1|-9] infile1 [infile2...]

Use -c to compress, -d to decompress, -t to test
 -h: don't fill or remove hi-res screen holes
 -9: high compression (default)
 -1: fast compression

Example: fhpack -c foo.pic foo.lz4fh



Usage: tohgr [options] a.xpm [options] b.png ... [options] z.jpg 

Options accepted:

-hgr   : convert following files to hires images
-dhgr  : convert following files to double hires images
-pack  : produce packed HGR/DHGR files (Filetype $08)
-3200  : convert following files to 320x200 super hires 3200 color images
-q [integer]  : quality value for super hires conversion (default = 16)
-atkin : use Atkinson dithering instead of Floyd-Steinberg dithering
-xpm   : output preview files in XPM format instead of PNG
-bmp   : output DHGR file in BMP format
